Getting started

After downloading the launcher and opening it for the first time, you will be greeted with a sign-in screen. In order to use the launcher, you need to log in.

Logging in

In order to log in, you need to pick an account type. Currently MojoLauncher supports the following account types:

  1. Local account

Local accounts are the most basic account type. They allow you to sign in using only the username, and do not perform authentication against any kind of service. Use a local account if you just want to play the game and don't plan on joining any online mode servers.

  1. Microsoft account

Mocrosoft accounts use the regular Minecraft authentication system. To successfully log in with a Microsoft account in MojoLauncher, you need to have Minecraft: Java Edition purchased on your Microsoft account, and a profile set up on https://minecraft.net

  1. Ely.by account

Ely.by accounts use the third-party Ely.by authentication system. Use this account type if you can't use a Microsoft account and want to see skins or play on a server that uses the Ely.by authentication system.

After logging in with your chosen account type, you can proceed to the launcher's main menu

Main menu

In the main menu of the launcher, you will find:

  1. The account selector (*). Click on if if you want to add more accounts to your launcher and change them.
  2. The settings menu (**). You will rarely need this, but if you want to learn more, visit the Settings guide
  3. The Wiki button, which will lead you to this website's main page in case if you need more information.
  4. The Discord button, which will lead you to the official MojoLauncher discord server.
  5. The Custom Controls editor, which allows you to customize your on-screen controls. For information on how to edit the control layout, visit On-screen control layout
  6. The JAR executor, which runs executable .jar files, like the OptiFine installer. Do not try to install mods through this button. To install mods, visit Installing mods
  7. The "Share log file" button, which allows you to easily share the log file for troubleshooting. For more information on troubleshooting, visit Troubleshooting
  8. The "Open game directory" button, which is used for installing mods, resourcepacks and other extras
  9. The instance selector (***), which allows you to choose and create new instances.
  10. The instance editor (****), which allows you to adjust the settings of your current instance.
  11. The "Play" button, which downloads and starts the game.
